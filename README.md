# Interactive Forecasting Tool

A Streamlit-based application for interactive forecasting.  
Users can upload their own datasets, set forecasting parameters, and instantly view results — without writing any code.

## Features
- 📂 **Data Upload**: Import CSV or Excel files easily.
- 🔍 **Exploratory Analysis**: Automatic summary statistics and visualizations.
- 📈 **Forecasting**: Apply time series forecasting models with adjustable parameters.
- 🎨 **Interactive Dashboard**: Results displayed in real-time with charts and tables.
- 🚀 **No-Code Experience**: Designed for users who want forecasting insights without programming.

## Tech Stack
- [Streamlit](https://streamlit.io/) for interactive UI
- Python libraries for data analysis and forecasting (pandas, statsmodels, etc.)

## Usage
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
